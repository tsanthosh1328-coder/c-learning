# c-learning

A collection of C programs written while learning the C programming language from scratch, following the **Bro Code C Tutorial Series** on YouTube.

---

## About

This repository contains my practice code for each lesson. I am learning C during my summer holiday, starting from the basics and working my way up.

**Tools used:**
- Editor: Visual Studio Code
- Compiler: GCC via MSYS2
- OS: Windows
